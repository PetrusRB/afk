## Features
*   ✅ **Auto-Connect**: automaticamente entra no servidor.
*   ✅ **Infinite Uptime**: previne expulsão de afk.
*   ✅ **Smart Reconnect**: automaticamente reconecta quando a internet/servidor cai.
*   ✅ **Render-Ready**: inclui "Self-Ping" que deixa o bot rodando por 24/7
*   ✅ **Plugin Support**: compativel com plugins Paper/Spigot/Bukkit (auto auth)
